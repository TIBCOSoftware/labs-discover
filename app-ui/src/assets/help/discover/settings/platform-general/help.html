Help Settings for Platform

Storage

This is where database settings are expected. The process miner job will store all the output there

Backend Type :  Storage type to be use as an ouput of the process mining job 

Values :

  • aurora (Postgresql Database)
  • sbdf (file format for TIBCO Spotfire)

Batch Size : The JDBC batch size, which determines how many rows to insert per round trip. This can help performance on JDBC drivers. This option applies only to writing. It defaults to 1000.

Values : 1000 - 100000

Partitions :  The maximum number of partitions that can be used for parallelism in table reading and writing. This also determines the maximum number of concurrent JDBC connections. If the number of partitions to write exceeds this limit, we decrease it to this limit by calling coalesce(numPartitions) before writing.

Values : 20-200

Database URL : The JDBC URL to connect to. The source-specific connection properties may be specified in the URL. e.g., jdbc:postgresql://localhost/test?user=fred&password=secret

Driver : The class name of the JDBC driver to use to connect to this URL.

Username :  Username used to get access to the database, needs dbo level

Password :  Password for the sql user set above

Communication

This is where messaging settings are expected. The process miner job will slisten to requests from there

Endpoint :  eFTL endpoint URL as shown inn the endpoint section of TIBCO Cloud Messaging 

Messaging Key : Access KEY to authentify against TIBCO Cloud Messaging

Config URL :  https endpoint for accessing the analysis configuration (public service for discovery in Flogo)

Analytics

This is where spotfire settings are expected.

Boolean :  enable this button if you wanna use the SaaS version of spotfire as part of your subscription 

Spotfire server URL : Spotfire url (https) to use for this application

Analysis lifecycle application

This is where LiveApps settings are expected.

App Name :  Live Apps application to be use for storing the metadata of analysis. 

Create Name : LiveApps creator to be used for creating a new entry.
