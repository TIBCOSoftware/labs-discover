Data sources and associated data formats can vary widely, but process mining only needs three key attributes:

  • a case ID (unique identifier)
  • a timestamp
  • an activity name
These three attributes represent “an activity performed at a point in time, as part of a single case or process”.

Once this information is provided, process mining analyzes, correlates, and reconciles this data , and provides graph-based views of the results to business process stakeholders, who can then start the exploration and insight discovery process.
