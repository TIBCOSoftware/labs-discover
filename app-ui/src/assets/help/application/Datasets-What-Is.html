A Dataset represents an event log data, connected to (or extracted from) your operational systems and databases

Each dataset should contain at least 3 columns representing the the following concepts

  • CaseId: unique identifier of a process instance.
  • Activity name: a description of an activity performed as part of a process instance.
  • Activity start timestamp: the date time at which the activity was executed.

    • In addition to the above minimun requirements, you can add as many additional columns as you want, in order to make your analysis more relevant.
