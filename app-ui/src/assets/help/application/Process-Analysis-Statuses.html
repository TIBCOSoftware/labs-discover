A process analysis can have any of these statuses.

  • Process Mining: The process mining algorithms have been sent to the backend for processing. A progress bar tracks the overall progress. After processing is complete, the analysis becomes ‘Ready’ or ‘Not Ready’.
  • Ready: The process analysis completed successfully and analytics are ready for viewing.
  • Not Ready: The process analysis terminated with errors. After the errors are corrected, the analysis can be rerun.
  • Archived: The process analysis is available for viewing but can’t be modified.
  • Deleted: The process analysis has been removed from the system.
