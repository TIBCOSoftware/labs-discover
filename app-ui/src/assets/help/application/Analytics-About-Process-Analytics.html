Process analytics for Discover are powered by Spotfire. Use the system templates as a starting point. The templates are designed to be generic so you can use them with any data source. They include default visualisations for Process Maps, Variants, Cases, and Resources.

Use the templates feature to assign specific analysis files (DXPs) to your process analysis, customize the menu, and add filters.

Leverage the power of TIBCO Spotfire, there's no limit !
