The possible actions depend on the status of the process analysis. The list of those actions is

  • Edit: allows to change the configuration of the analysis (eg. name, descirption, mapping...).
  • Rerun: allows to re-execute the process mining algorithm in the backend.
  • Abort: allows to interrupt the execution of process mining.
  • Archive: allows to mark the analysis as readonly.
  • Delete: allows to remove permanently the analysys and it's data.
