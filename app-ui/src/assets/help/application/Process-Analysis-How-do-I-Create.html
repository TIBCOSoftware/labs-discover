To create a process analysis:

  1. On the Process Analysis page, click + Analysis.
  2. Follow the wizard steps to create a new entry in the process analysis table. For help on the steps, click the help icon when you’re in the wizard.
  3. Track the lifecycle of the process analysis until it reaches the Ready state.
  4. To choose a visualization template that’s suited for your dataset and analysis, click Select Template. You can also create your own template.
  5. To start visual process mining, click View Analytics on the Process Analysis page.
