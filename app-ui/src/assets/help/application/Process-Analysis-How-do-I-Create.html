To create a Process Analysis:

  1. Click + Analysis.
  2. Follow the steps of the Wizard in order to create a new entry in the Process analysis table
  3. Track the Lifecycle of the Process analysis, until it reaches the Ready state
  4. Click on "select Template" to choose the best template suited for your dataset and analysis. Remember you can also create your own!
  5. Click on "view Analytics" to start visual process mining
