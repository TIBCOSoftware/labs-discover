This set of visualizations focus on resources. Also known as social mining. The purpose is to help discover some patterns related to how resources are involved and interact with each other in processes.

A table lists resources ranked by number of activities performed.

  • select one or more resources to see details, including the list of activities performed by the selection.

    • The graph represent resources as nodes and connections as edges. A connection between two resources represents two consecutive activities in the same case. Of course, an edge can represent more than one occurrences of such connection, for one or more cases.

    A filter panel allows

  • setting a minimum number of activities, as a way to filter resources
  • setting a minimum number of activities, as a way to filter resources and connections
