This set of visualizations, also known as social mining, focuses on resources. The purpose is to discover patterns related to how resources are involved and interact with each other in processes.

The table lists resources ranked by number of activities performed. Select one or more resources to see details, including the list of activities performed by the selected resource.

The graph represents resources as nodes and connections as edges. A connection between two resources represents two consecutive activities in the same case. An edge can represent multiple connections for one or more cases.

The filter panel allows you to filter the resources and connections.
