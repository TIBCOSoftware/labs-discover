Note that if your dataset contains any of the following column names:

  • input_case_id
  • input_activity_id
  • input_activity_start_timestamp
  • input_activity_end_timestamp
  • input_resource_id
  • input_resource_group
  • input_requester
  • input_scheduled_start
  • input_scheduled_end
    • Then they will automatically be mapped to be used for process mining. You will not be able to change those mappings and will see them as "locked" in the mapping screen. If you want to use a different column for one of these mappings, please rename the column that matches the list above, and then your mapping will be unlocked and free to change.
