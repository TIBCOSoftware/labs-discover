You will use TIBCO Cloud Live Apps to create or edit case management applications that will implement your investigations.

Live Apps is a very simple service that requires no coding. how does it work ?

  • First, you will use Live Apps Designer to author the application. Creating a new app is a matter of minutes, and you can use the examples as a starting point. Your applications is structured around states, data and actions.
  • Second, you can test the application ether from Live Apps Designer or using Live Apps Case Manager
  • third, you will use Live Apps Administrator to publish the application and make it ready to use by users.
  • Finally, once published, you can associate it with Discover as a new investigation type.
