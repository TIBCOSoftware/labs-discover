Use TIBCO Cloud Live Apps to create or edit case management applications that implement your investigations.

  1. Use Live Apps Designer to create the application. Each application is structured around states, data, and actions. Applications can be created quickly and examples help get you started.
  2. Test the application from Live Apps Designer or using Live Apps Case Manager.
  3. Use Live Apps Administrator to publish the application and make it ready to use by users.

After your app is published, you can associate it with Discover as a new investigation type.
