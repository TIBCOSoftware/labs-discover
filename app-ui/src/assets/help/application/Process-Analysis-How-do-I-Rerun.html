When an analysis is in the ‘Not ready’ state, it means that something prevented it from running the process mining job successfully.

In such a situation, we recommend the following:

  • Read the error message by hovering around the "Not ready" status column for the analysis.
  • Read the help section on most common mistakes and see whether some solutions are suggested.
  • Fix the underlying problem, which is often related to the dataset or mappings.
  • Rerun the analysis. On the Process Analysis page, click the More icon (...) and select Rerun.
