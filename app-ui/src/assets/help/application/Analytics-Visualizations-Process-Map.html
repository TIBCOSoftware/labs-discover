Process Map is a graph representing your processes. Whether it shows one ore more variants, the purpose is to help you visualize the end-to-end flow.

Typical features include

  • Node color represent a metric (frequency or performance)
  • Zoom in and out to explore the process
  • Support of multiple layouts
  • Ability to select nodes and edges for filtering purpose
  • Configurable labels on edges (including Nb events & duration)
