Click + Dataset and follow the wizard steps.

  • Basic Info: Enter a name for the dataset. The name must be unique across all your datasets.
  • Data source: Use a CSV file or TIBCO Data Virtualization (TDV) view.
    • CSV: Upload a CSV file or select one from the list of available files.
    • TDV: Select one of the available views.
  • Preview: View a preview of the data, if available. If the data isn’t what you want to work with, click Previous and select or upload another file.
  • Attributes: This step displays the results of the system’s attempt to determine the data types of the uploaded data. In this step, review the data types that the system selected and make any changes. The Value Preview column contains the values from the first row in the dataset.
  • Dates: The system tries to figure out the format of the date fields in the imported data. Review the format and change it if needed. Here also you can provide your own format, please refer to this link for more information
  • Confirmation: Review the dataset settings. To make any changes, click Previous and navigate to the step you want to revisit.
