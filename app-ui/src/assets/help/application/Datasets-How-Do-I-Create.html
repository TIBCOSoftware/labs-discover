To Create a new Dataset, click on the Create button and follow the steps of the wizard

  • In Basic Info, enter a name and description for the dataset.
    • Note: names needs to be unique across all datasets.
  • Data source: Choose between Uploading a CSV File or connecting a TIBCO Data Virtualisation view
    • then either select a file using the file browser, or select a view from the drop down coming from the TDV catalog.
  • Preview: Here you can see a sample preview of your data, to make sure it looks alright.
  • Attributes: The system will try to automatically figure out the types, and will leave you an opportunity assign them as well in case of mistake. You have the last word!
  • Dates: The system will try to automatically figure out the date formats for each fo the date fields. Here also you can provide your own format, please refer to this link for more information
  • Confirmation: A summary of our dataset is displayed before you can confirlm and create it in the system.
