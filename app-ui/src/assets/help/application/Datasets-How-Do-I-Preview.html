Datasets can be very large, depending on the underlying data source. The Preview feature allows you to do the folllowing :

  • During the creation or Editing of dataset, you can preview a sample of the data, typically the first 50 lines
  • During the Creation of the new Process analysis, you can access a data summary dashboard, including data distributions and metrics.

    • At anytime, you can refresh the preview, in order to match the latest data, especially if the underlying data source is a data view connected to external systems.
